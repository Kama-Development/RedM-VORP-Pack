# RedM-Grave-Robbing
Adds the ability to rob gravestones in RedM.

**Dependencies**



**Instructions**

- Extract download into your resources folder
- Start the resource in your server.cfg


  "oldbuckle",
  "oldwatch",
  "goldring",
  "goldtooth",
  "oldcoin",
  "robbingkit"
  
- In-game, use a "robbingkit" near a gravestone to rob it
