# vorp_loot mod
CryptoGenics loot mod 

Modified to work better with VORP, added loot message and XP loot for burglars profit :)

# Installation
1. Extract the zip.
2. Put vorp_loot_mod to your resource folder.
4. Add "ensure vorp_loot_mod" in your "server.cfg".
5. Profit

# Required resource
- VORP Core
